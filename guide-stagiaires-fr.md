## Programme jour par jour (rappel)
1. Accueil, choix finaux d'options, premiers assemblages
2. Structure bois (MD/MG), ponçage, collage, équerrage, base mécanique
3. Claviers & boutons : axes, ressorts, toucher
4. Mécaniques MD/MG : assemblage complet, réglages hauteurs/courses
5. Soufflets : fabrication, pose, ajustements, étanchéité
6. Grilles & finitions : montage, décor, finitions bois
7. Anches : tri, installation, mise en tension, tests note par note
8. Réglages fins : fuites d'air, mécanique, alignement touches
9. Tests complets MD+MG, confort de jeu, corrections personnalisées
10. Pratique musicale avec intervenant, test final, photos, bilan collectif

## Journée type
- 9h30–12h30 : atelier fabrication
- 12h30–14h00 : pause déjeuner
- 14h00–17h30 : montage, réglages, accompagnement individuel

## Budget hébergement proche atelier (9 rue Fernand de Magellan)
- Airbnb centre / proche mer (studios, petits apparts, chambres privées) : filtre "max 60 €/nuit", 15–30 min à pied.
- Exemples types :
  - Studio simple proche plage/commerces (~50–80 €/nuit)
  - Petit T2 avec kitchenette (~50–70 €/nuit)
  - Chambres privées économiques (~17–40 €/nuit)
- Hôtels budget (ibis budget, B&B) : centre/gare ~15–25 min à pied
- Résidence jeunes / auberge : ~15–30 min à pied
- Campings en bus/covoiturage 15–30 min : Saint-Brevin-les-Pins, Pornichet, Batz-sur-Mer, Préfailles, Guérande, La Baule  
  _(à compléter avec adresses exactes + temps de trajet vérifiés)_
- Conseils : vérifier frais ménage/service Airbnb, privilégier options sans voiture, filtre "proche centre/plage/transport".
